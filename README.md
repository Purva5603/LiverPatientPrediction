#  Liver Patient Prediction using Machine Learning

##  Project Overview
This project aims to predict whether a patient is affected by **liver disease** based on clinical and demographic data. Early detection of liver disease can help in timely diagnosis and better treatment planning.

A **Logistic Regression model** was selected as the final model due to its strong performance, simplicity, and interpretability in medical prediction tasks.

---

##  Objective
- Predict liver disease presence in patients  
- Perform exploratory data analysis to understand key health indicators  
- Build and evaluate machine learning models  
- Select the best-performing and interpretable model  

---

##  Dataset
- **Dataset Name:** Indian Liver Patient Dataset (ILPD)  
- **Source:** UCI Machine Learning Repository  
- **Target Variable:** Liver Disease Status (Patient / Non-Patient)  

### Dataset Features Include:
- Age  
- Gender  
- Total & Direct Bilirubin  
- Alkaline Phosphotase  
- Alamine Aminotransferase (ALT)  
- Aspartate Aminotransferase (AST)  
- Total Proteins  
- Albumin  
- Albumin and Globulin Ratio

---

##  Model Used
### **Logistic Regression**
- Chosen for its:
  - High interpretability in healthcare applications  
  - Stable performance on structured data  
  - Ease of deployment  

---

##  Model Evaluation
- Evaluated using:
  - Accuracy  
  - Precision  
  - Recall  
  - Confusion Matrix  

Logistic Regression provided a good balance between performance and interpretability, making it suitable for this medical prediction task.

---

##  Key Insights
- Liver enzyme levels play a crucial role in predicting liver disease  
- Data preprocessing and feature scaling significantly improved model performance  
- Logistic Regression performed competitively compared to other models
